# ArxivGrab - AI Web Agent Chat Interface

### 前端 (React + Vite)

```bash
npm install

npm run dev

npm run build
```

### 後端 (Python Web Agent API)

```bash
pip install -r requirements.txt

export GOOGLE_API_KEY="your_google_api_key_here"

python start_api_server.py
```

### 同時運行前後端

```bash
python start_api_server.py

npm run dev
```
